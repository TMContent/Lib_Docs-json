Updating TEAM Mentor Eclipse Plugin for Fortify is as straight-forward as upgrading any Eclipse plugin.

  1. In the Eclipse toolbar click on Help -> Check for Updates
    After Eclipse contacts all the plugin repositories you will see a list of available updates. In this case only the TEAM Mentor Eclipse Plugin for Fortify has a new version and can be updated. Select the plugin and click “Next”
  2. Confirm the items update details and click “Next”.
  3. Accept the license and click “Finish”
  4. Software will be updated
  5. Restart Eclipse when prompted
