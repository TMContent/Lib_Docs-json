1. In the Eclipse toolbar click on Help -> Install New Software and click on “Add”




  2. In the dialog box that appears enter a name (TEAM Mentor Eclipse Plugin for Fortify) in the name section and for Install Link use:

    https://eclipse.TEAMMentor.net

    Click “OK”




  3. Select Security Innovation TEAM Mentor Plugin for Fortify and click “Next”




  4. Confirm the items being installed. Click “Next”




  5. Accept the license and click “Finish”




  6. Restart Eclipse when prompted
