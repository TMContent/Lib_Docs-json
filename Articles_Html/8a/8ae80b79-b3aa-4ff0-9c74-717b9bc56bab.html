Jade is the fail safe version of TEAM Mentor. TEAM Mentor was designed to run on modern browsers to provide a smooth user experience. However, we do realize that sometimes customers for various reasons cannot run newest browsers. To accommodate we have created the Jade version. While not as slick as the main, Flare, version it still provides the main functionality in an easy to use package. It runs on legacy browsers, including IE8 and Lynx. TEAM Mentor automatically determines if it your browser can't handle Flare version and will present you with Jade. Chances are you will never get to see it, but in case you do, this section of the documentation explains how to use it.
