Open Fortify Perspective and Findings
In this example we have a project with a simple, vulnerable, application.

  • Analyze the project by going to HP Fortify->Analyze Project
    or
  • Open an already saved analysis by going to HP Fortify -> Load Saved Audit Project or HP Fortify -> Open Audit Project
    This brings up the Fortify Audit perspective


As you can see TEAM Mentor guidance is automatically displayed whenever a Fortify finding is selected in the SCA Analysis Results view.
